# Peak Beacom - Engineering Certification Program
## Capstone Project

This GitHub repository is for owners of their very own AutaWata to update it.

In order to properly update your AutaWata, you will need to install the Arduino IDE.

Windows
https://downloads.arduino.cc/arduino-ide/arduino-ide_latest_Windows_64bit.exe


macOS
https://downloads.arduino.cc/arduino-ide/arduino-ide_latest_macOS_64bit.dmg

Linux
https://downloads.arduino.cc/arduino-ide/arduino-ide_latest_Linux_64bit.AppImage


After installing the IDE on your laptop, you can open the correct file for your version of AutaWata, connect your AutaWata with the provided cable, select the "Mega or Mega 2560" option if one pops up. After selecting the AutaWata's board, click the arrow to update your AutaWata. The on-board data memory will save, and you do not have to re-complete the setup process.


The AutaWata website will provide you with extra information on the AutaWata.


https://peakbeac.github.io/ecp/autawata
